# k线因子

我希望将“量化级别的K线形态因子”与“缠论结构”结合，我们就需要把这两套体系彻底打通。缠论提供的是“走势结构的骨架”（定区域、定级别），而K线形态提供的是“多空博弈的血肉”（定力度、定时机）。
下面我为您全景梳理这两大体系的核心构件，并逐一拆解它们在实际交易中代表的具体信号。

## 缠论结构因子：走势的几何骨架
缠论的核心思想是“走势终完美”，它通过严格的几何递归，将市场走势从最小零件组装成完整建筑。其核心构件代表的信号如下：
1. 分型（顶分型 / 底分型）
形态定义：由处理完包含关系的相邻三根K线组成。顶分型中间K线高低点均最高，底分型中间K线高低点均最低7。
代表信号：潜在转折信号。它是市场转折的最小单元，预示价格可能由涨转跌或由跌转涨。但分型本身不是绝对的买卖点，必须结合位置（如关键支撑阻力位）和力度（如放量大实体）来确认其有效性37。
2. 笔
形态定义：连接相邻的顶分型和底分型，中间至少包含一根独立K线，构成走势的最小运动单元。
代表信号：方向确认信号。向上笔（底到顶）代表短期多头主导，向下笔（顶到底）代表短期空头主导。笔的延伸与破坏，是判断短期趋势是否延续的最直观信号7。
3. 线段
形态定义：由至少三笔构成，且前三笔必须有重叠部分。线段比笔更稳定，是构成中枢的核心单元7。
代表信号：中期方向信号。线段代表了次级别的趋势方向，只有当线段被反向线段彻底破坏时，才确认中期趋势的终结7。
4. 中枢
形态定义：某级别走势中，至少三个连续次级别走势重叠的价格区间。它是多空力量博弈的“战场”和筹码密集区37。
代表信号：走势分类与操作导航信号。中枢定义了走势类型：只包含一个中枢为“盘整”，包含两个及以上同向中枢为“趋势”。中枢的延伸、新生、扩展，直接决定了是该持股观望、逢低吸纳还是果断止损35。
5. 背驰（趋势背驰 / 盘整背驰）
形态定义：价格创出新高或新低，但推动力度（如MACD黄白线或柱子面积）却出现衰竭35。
代表信号：趋势终结信号（核心买卖点）。背驰是缠论寻找第一类买卖点（抄底逃顶）的核心依据。趋势背驰预示大级别反转，盘整背驰预示小级别震荡的高低点3。

## 量化级别K线形态因子：情绪的微观刻画

在量化回测中，K线形态被剥离了主观色彩，通过实体占比、影线比例、ATR（真实波幅）等参数进行严格过滤。以下是经典形态的量化信号拆解：
1. 看涨反转信号（寻找买点）
早晨之星 / 晨星：大阴线 + 小实体/十字星 + 大阳线（切入前阴线实体50%以上）。信号：下跌动能彻底衰竭，多头强势反攻，是极佳的波段低吸确认信号6。
看涨吞没（阳包阴）：阳线实体完全包裹前一根阴线实体。量化信号：要求阳线实体占比高，且K线总范围大于1.2倍ATR。代表买方在一天内完全占据主导，反转可靠性高1。
曙光初现（刺穿线）：大阴线后，阳线跳空低开但收盘深入阴线实体一半以上。信号：空头打压遇阻，多头开始反扑，常作为回调结束的预警6。
锤子线 / 倒锤头：长下影线（或上影线）伴随小实体，出现在下跌末端。信号：下方有极强买盘支撑，空头力量耗尽6。
2. 看跌反转信号（寻找卖点/止盈点）
黄昏之星 / 晚星：大阳线 + 小实体/十字星 + 大阴线（切入前阳线实体50%以下）。信号：上涨动能枯竭，空头开始主导，是波段高位减仓或清仓的强烈信号6。
看跌吞没（阴包阳）：阴线实体完全包裹前一根阳线实体。信号：卖方力量呈压倒性优势，趋势可能由涨转跌6。
乌云盖顶：大阳线后，阴线跳空高开但收盘深入阳线实体一半以下。信号：多方上攻乏力，高位抛压沉重，是典型的见顶预警6。
射击之星 / 上吊线：长上影线伴随小实体，出现在上涨末端。信号：上方阻力极大，多头无力维持高位，随时可能回落6。
3. 趋势中继信号（寻找加仓点）
上升三法 / 红三兵：连续三根阳线（或大阳线后接三根不破其低点的小阴线，再收大阳线）。信号：上涨趋势中的良性洗盘或稳步推升，代表趋势延续6。
下降三法 / 三只乌鸦：连续三根阴线（或大阴线后接三根小阳线，再收大阴线）。信号：下跌趋势中的弱势反弹，随后将继续主跌浪6。

## 进阶：两套体系的“共振”实战应用
在成熟的量化策略中，这两套体系绝不是孤立使用的，而是“缠论定区域，K线定时机”：

谐波形态 + K线形态：通过斐波那契比率（如Gartley、Bat形态）精确计算出潜在的反转区域（D点），然后必须等待D点附近出现K线确认信号（如看涨形态对应锤子线、吞没阳线）才作为入场信号2。

缠论买卖点 + K线确认：当缠论在日线级别识别出“底背驰”或“底分型”时，不盲目左侧接飞刀，而是切入次级别（如30分钟或5分钟），等待次级别出现“看涨吞没”或“早晨之星”等放量K线形态后，再执行右侧买入23。
